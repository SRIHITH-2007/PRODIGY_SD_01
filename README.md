# TempConverter
Title: Temperature Converter (Task 1 – Software Development Internship)
This program converts temperatures between Celsius, Fahrenheit, and Kelvin based on user input.

Features:
> Convert from Celsius → Fahrenheit & Kelvin
> Convert from Fahrenheit → Celsius & Kelvin
> Convert from Kelvin → Celsius & Fahrenheit
> Handles invalid unit input

Sample Output: 
        Enter temperature value: 25
        Enter unit (C/F/K): C

        --- Converted Temperatures ---
        Celsius: 25.0
        Fahrenheit: 77.0
        Kelvin: 298.15
